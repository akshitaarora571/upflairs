git commands----

----create folder or initialize git repo
git init

----status 
git status

----username and email config
git config user.name " "
git config user.email " "

----staging the file
git add filename,filename
##or
git add .-----for all

----commit the changes
git commit -m "any message"

----one time step--adding remote to your git--
git remote add origin "http://url_of_your_repo"

----git clone"the_other_repo"
----master
git checkout master
git checkout -b -----nayi_branch

----push the changes to branch
git push origin nayi_branch

----pull content from other branch
git pull origin master

----merge the changes in other branches
git merge

----fetch all the tree structure from remote
